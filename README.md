# vftraining
Repository for implementing a first application for training purposes

## Alarm viewer
To train the VF team on MW SDN Application Layer concepts, a small application has been designed to display the current alarms of devices in the MW network.

**Purpose:**
This application serves as a practical example to help the team to, 
- Understand the Application Pattern concept.
- Recognize the importance of integrating new applications with the Tiny Application Controller.
- Learn how to interpret the application specification and configuration files developed in alignment with ONF principles.
